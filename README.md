# myITS Login Script

## Usage
1. Copy `config.sample.py` to `config.py`.
2. Run `config.py` and enter the output to password in `config.py`.
3. Run `main.py`.
4. Copy `TVMSESSID` cookie to use the session, use Postman or Cookie Editor browser extension.
